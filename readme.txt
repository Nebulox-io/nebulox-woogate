Nebulox Crypto Gateway enables WooCommerce stores to accept cryptocurrency payments.

== Description ==

**Nebulox** is a cryptocurrency payment gateway designed for WooCommerce, allowing merchants to accept payments in various cryptocurrencies. The plugin supports USD and EUR, with automatic conversion for other currencies via the Nebulox API. It features an easy-to-use interface for configuring settings such as the API key, webhook URL, and redirect URL.

= Key Features =
* Accept cryptocurrency payments on your WooCommerce store.
* Supports USD, EUR, and conversion for other cryptocurrencies.
* Seamless integration with WooCommerce payment settings.
* Easy setup for API key, webhook, and redirect URLs.

== Installation ==

1. Upload the `nebulox` plugin folder to your `/wp-content/plugins/` directory, or install the plugin via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > Settings > Payments and enable the Nebulox Crypto Gateway.
4. Enter your API key and configure the plugin settings as needed.

== Frequently Asked Questions ==

= What is Nebulox? =

Nebulox is a cryptocurrency payment gateway that integrates with WooCommerce to accept crypto payments.

= How do I obtain an API key? =

You can register and obtain an API key from our [official website](https://nebulox.io).
Once logged in, navigate to the "Gateway" section via the sidebar.
Create a new gateway and complete the required fields.
After successfully creating the gateway, you will receive an API key.

= Which currencies are supported? =

The Nebulox plugin supports payments in USD and EUR, with automatic conversion for other cryptocurrencies through the Nebulox API.

= How do I configure the plugin? =

Once installed and activated, go to WooCommerce > Settings > Payments, enable Nebulox Gateway, and configure your API key and settings.
